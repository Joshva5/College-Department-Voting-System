📌 About the Project

VoteCampus is a secure and real-time web-based voting system designed to manage college department elections efficiently. The platform allows students to register, log in, and cast their votes for various roles such as President, Vice-President, and other leadership positions.
The system ensures transparency, fairness, and security by enforcing one vote per student per role and providing live election updates without refreshing the page.
An Admin panel is included to manage the entire election process — from adding candidates to controlling election timing and monitoring live results.

🚀 Key Features

🔒 Secure Authentication using JWT
🗳️ One Student – One Vote per Role
⚡ Real-Time Voting Updates using WebSocket
📊 Live Results Dashboard
👨‍💼 Admin Panel for Candidate & Election Management
⏳ Election Timer (Auto Start & Stop)
📢 Live Announcement System
📱 Responsive UI for better user experience


🧠 How It Works

Students register and log in securely
Admin creates roles and adds candidates
Admin starts the election with a timer
Students vote for candidates (only once per role)
Votes update instantly using WebSocket
Results are displayed live on the dashboard
